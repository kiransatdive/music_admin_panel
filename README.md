# Admin Panel

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm preview
```

## Features

- Admin authentication with JWT
- Dashboard with platform statistics
- Release approval and management
- Whitelist management (artists, labels, distributors)
- Content Management System (CMS) for public website
- Payment report management and export
- YouTube monetization criteria configuration
- Platform-wide revenue analytics
- Notification system for users
- PWA support for offline access

## Pages

- Login - Admin login page
- Dashboard - Overview with stats and quick actions
- Releases - Review and approve/reject releases
- Whitelist - Manage whitelisted entities
- CMS - Update public website content
- Payments - Manage payment reports and exports
- YouTubeCriteria - Configure YouTube monetization tiers
- Revenue - Platform revenue analytics
- Notifications - Send system notifications

## PWA Installation

The admin panel is configured as a PWA. Users can install it:
- On desktop: Click the install icon in the browser address bar
- On mobile: Use the "Add to Home Screen" option from the browser menu
